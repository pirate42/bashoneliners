O hi, we aim to collect here *good* bash/ksh one-liners. We are in a very very early stage of development but we will progress rapidly.

Features in the making

  • Create account, login. Members will be able to post one-liners.
    • Improve the ugly interface.
    • Profile page for logged in members with list of own posts and other goodies.
  • Vote on a one-liner. Members will be able to vote up or down a one-liner.
  • Comment on a one-liner. Members will be able to comment on one-liners and make suggestions.

If you would like to participate in the development or if you would like an early member access to contribute one-liners, please let us know!
